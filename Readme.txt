" *There can be life after
cancer. The only prerequisite being early detection. *"

https://my.visme.co/view/8r4097rq-m3x58k4ko9pk5krp

Creating an assistive software that helps doctors identify tumors faster and classify the type of cancer (benign or malignant) and stage. Also notify the patient about the results and provide measures to be taken.
The technology used in the software is machine learning and neural networks. 
Testing data is sourced from The Cancer Internet Archive.
It uses image processing techniques to analyze the samples given in the form of images and circles the identified tumors if any so that the doctor's time is not wasted looking for it.
